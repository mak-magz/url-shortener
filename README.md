# URL Shortener

A URL shortener service — built as a DevOps practice project.

## Goals / Tech Stack

### Phase 1 (Fullstack)
- Frontend
  - Vue.js / Nuxt.js
	- Tailwind CSS
	- Pinia
	- TypeScript
	- Nuxt UI
- Backend
  - Go (Gin)
	- PostgreSQL / pgx
- Docs
  - Markdown
	- Swagger

### Phase 2 (Docker)
- Frontend Dockerfile
	- Multi-stage build
- Backend Dockerfile
	- Multi-stage build
- Docker deployments
	- Docker Compose
- Reverse Proxy
	- Nginx

### Phase 3 (CI/CD)
- GitHub Actions workflows
- Terraform deployments

### Phase 4 (Kubernetes)
- Kubernetes deployments
- Helm deployments

### Phase 5 (Production)
- Monitoring
	- Prometheus
	- Grafana
- ArgoCD
	- ArgoCD configs
- Terraform
	- Terraform modules
